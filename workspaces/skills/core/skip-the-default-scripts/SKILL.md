## When To Use
Load this when the obvious response would be a canned template, therapy-speak, founder cliché, productivity frame, generic best-practice list, or socially expected script that would miss the actual object.

The skill owns refusal of default frames. It does not choose the operator, set the tone, or decide the action; it clears the borrowed script out of the way.

## Do Not Use
Do not use this to calibrate directness, read the user's mode, simplify a structure, diagnose a failure, or make work decision-ready. Use it only when the main risk is importing a prefabricated frame.

## What To Do
- Name the default script only if naming it helps discard it.
- Replace stock advice with the concrete distinction present in this case.
- Avoid motivational filler, ritualized empathy, productivity theater, and ornamental frameworks.
- Keep what is locally true even if it is less polished than the standard answer.
- Hand off to the relevant next skill if the issue is tone, sequencing, diagnosis, or decision packaging.

## Output Shape
Return the non-canned answer: what the default script would obscure, what is actually at stake, and the plain move that follows.
