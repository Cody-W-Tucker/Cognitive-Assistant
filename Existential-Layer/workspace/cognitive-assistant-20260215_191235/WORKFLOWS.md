## Recurring Playbooks

### Playbook: When Stuck in Reversible Decision
1. Notice the paralysis pattern (irreversible decisions don't trigger this)
2. Ask: "What is my gut already saying?"
3. Identify the fear driving the hedging
4. Take smallest possible action in gut direction
5. Let feedback inform, not paralyze

### Playbook: When Tempted to Pivot
1. Check: Have I completed 3 iterations on current path?
2. Distinguish: Is this genuine signal or discomfort with execution phase?
3. If genuine signal: Pivot with intention
4. If execution discomfort: Find operator to handle execution

### Playbook: Evaluating Collaboration Fit
1. 90-day explicit evaluation point (non-negotiable)
2. Assess: Can they match my pace on their domain?
3. Assess: Do they handle execution I don't want to?
4. If no to either: Exit gracefully, don't over-invest

### Playbook: Packaging Ideas for Reach
1. Reframe: "Making wisdom accessible" not "dumbing down"
2. Start with existing writing archive
3. Extract one transferable principle per piece
4. Test with one trusted person outside his bubble
5. Iterate packaging, not core insight
